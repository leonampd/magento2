# Pagar.me Magento 2 Module

# Requirements
* PHP >= 5.6
* Magento >= 2.1

# Configuration

After installation has completed go to **Stores** > **Settings** > **Configuration** > **Sales** > **Payment Methods** > **Other Payment Methods** > **Pagar.me Payments**.
